## What is Space Birds?

Space Birds is an exciting arcade-style shooter game built with the libGDX framework. Take control of a brave bird defending against waves of falling pig obstacles in this Angry Birds-themed adventure.

The game features intuitive controls, engaging gameplay mechanics, and a vibrant visual and audio experience that brings the classic Angry Birds theme to life in a new format.

## Game Overview

Space Birds features three distinct game states:

- **Menu** - Start screen with game controls information
- **Playing** - Active gameplay with obstacle spawning and shooting mechanics
- **Game Over** - End screen displaying your final score

The gameplay is simple yet addictive: control your bird character, shoot down falling pig obstacles, and survive as long as possible to achieve the highest score. Every second you survive adds to your score, and each pig you destroy grants bonus points.

## Built with libGDX

Space Birds is powered by [libGDX](https://libgdx.com/), a cross-platform Java game development framework. This allows the game to run seamlessly on multiple platforms while maintaining high performance and native feel on each platform.
